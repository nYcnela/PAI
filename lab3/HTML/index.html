Podstawowe znaczniki HTML to:

  • <html> - korzeń dokumentu
  • <head> - nagłowek strony z informacjami dla przeglądarek
  • <body> - właściwa treśc dokumentu widoczna w oknie pregladarki
  • <div> - element blokowy
  • <p> - akapit
  • <ol>, <ul>, <li> - elementy listy wypunktowanej lub numerowanej
  • <img> - obrazek
  • <a> - hiperłącze

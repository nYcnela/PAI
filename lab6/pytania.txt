<======Odpowiedzi======>  

1. 
    1)Istnieje kilka sposobów na osadzenie skryptów JavaScript w dokumencie HTML
    Tag <script> - Najczęściej stosowaną metodą jest tagu <script> w sekcji head lub body dokumentu HTML. Można osadzić skrypt bezpośrednio w tagu lub wskazać na plik zewnętrzny poprzez atrybut 'src'  

    2) Atrybut 'onclick' - Można umieścić kod JavaScript bezpośrednip w atrybucie 'onclick' elementu HTML który zostanie wywołany po kliknięciu  

    3) Atrybut 'onload' - Atrybut 'onload' pozwala na wykonanie kodu JavaScript po załadowaniu elementu HTML np obrazka  

    4) Atrybut 'defer' lub 'async' - Można dodać atrybut 'defer' lub 'async' do tagu <script> aby opóźnić lub asynchronicznie załadować skrypt. Atrybut 'defer' pozwala na załadowanie skryptu asynchronicznie, ale wykona go dopiero po załadowaniu całej strony. Atrybut 'async' natomiast pozwala na asynchroniczne ładowanie i wykonanie skryptu nie czekając na załadowanie całej strony

-------------------------------------------------------------

2. 
    W dokumencie HTML można użyć wielu atrybutów elementów do obsługi zdarzeń, oto kilka najczęściej używanych 
    1) 'onclick' - wywołuje funkcję JavaScript po kliknięciu na element
    2) 'onmouseover' - wywołuje funkcję JavaScriptm gdy kursor myszy zostanie umieszczony na elemencie
    3) 'onload' - wywołuję funkcję JavaScript, gdy element zostanie załadowany
    4) 'onsubmit' - wywołuje funkcję JavaScript gdy formularz zostanie przesłany
    5) 'onkeydown, onkeyup, onkeypress - wywołują funkcję JavaScript, gdy zostanie naciśnięty lub zwolniony klawisz na klawiaturze

-------------------------------------------------------------

3.
    DOM - Document object Model reprezentacja dokumentu HTML lub XML w postaci hierarchi obiektów JavaScript. Model DOM umożliwia programistom dostęp do poszczególnych elementów dokumentu i ich właściwości, co pozwala na manipulowanie dokumentem za pomocą JavaScript
    Model DOM składa się z drzewa węzłów, w którym każdy element dokumentu reprezentowany jest przez węzeł. Węzły są połączone relacjami rodzic-dziecko, które odpowiadają strukturze dokumentu HTML. Na przykład, element body jest dzieckiem elementu html, a element p jest dzieckiem elementu body.

    Model DOM umożliwia programistom modyfikowanie elementów dokumentu, np. zmianę tekstu w polach formularza, dodawanie i usuwanie elementów, zmianę stylów CSS, a także dodawanie i usuwanie obsługi zdarzeń. Dzięki temu można tworzyć interaktywne strony internetowe, które reagują na akcje użytkownika.

    Istnieją dwa modele DOM: model DOM dla HTML i model DOM dla XML. Model DOM dla HTML jest bardziej popularny i jest używany do manipulowania dokumentami HTML, natomiast model DOM dla XML jest używany do manipulowania dokumentami XML i XHTML

4. Podaj podstawowe metody JavaScript  

    1)getElementById() - zwraca element dokumentu HTML o podanym identyfikatorze
    2)querySelector() - zwraca pierwszy element dokumentu HTML pasujący do podanego selektora CSS
    3)addEventListener() - dodaje obsługę zdarzenia dla elementu dokumentu HTML
    4)setAttribute() - ustawia wartość atrybutu dla elementu dokumentu HTML
    5)appendChild() - dodaje nowy węzeł jako dziecko elementu dokumentu HTML
    6)innerHTML - zwraca lub ustawia kod HTML dla elementu dokumentu HTML
    7)parseInt() - konwertuje ciąg znaków na liczbę całkowitą
    8)parseFloat() - konwertuje ciąg znaków na liczbę zmiennoprzecinkową

Podstawy Aplikacji internetowych

Aplikacja internetowa to zbiór plików i związanych z nimi komponentów, składowanych w określonym katalogu i jego podkatalogach na serwerze WWW. Aplikacje webowe działają według modelu wielostronnicowego (klasycznie) lub modelu jednostronnicowewgo (SPA - single page aplication). POdstawowymi językami do budowy aplikacji internetowych, interpretowanymi przez przeglądarke są: HTML, CSS i JavaScript.

HTML 5

HTML to podstawowy język definicji dokumentów dla klientów HTTP. Umożliwia zapis treści dokumentów i równocześnie opis jego układu za pomocą odpowiednich znaczników. Odpowiada za strukturę tworzonej strony internetowej i poszczególnych dokumentów.

Kurs HTML5

CSS

CSS odpowiada za wizualną prezentację stron internetowych w przeglądarkach. Arkusz CSS jest kolekcją reguł formatujących, które mogą odnosić się do wielu dokumentów HTML. Spełnia funkcję wzorca i pozwala na zapewnienie takiego samego wyglądu wszystkim wystąpieniom danego elementu.

Kurs CSS

JavaScript

JavaScript dodaje do stron WWW interaktywność. Umożliwia reagowanie na polecenia użytkownika, sprawdzanie poprawności formularzy, obsługę elementów interfejsu strony, dynamiczne generowanie treśći itp. Jesto to język zorientowany obiektowo, składnia i podstawowa struktura wzorowana jest na C/C++.

Kurs JavaScript

React

React to biblioteka JavaScript do tworzenia interfejsów użytkownika. Pozwala na łatwe tworzenie dynamicznych aplikacji internetowych z wieloma stanami widoku.
